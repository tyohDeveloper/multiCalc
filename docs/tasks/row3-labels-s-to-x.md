# Row 3 key labels (S–X)

## What & Why
Update the display labels for the six Row 3 buttons (s–x) to the correct values specified by the user.

## Done looks like
- Button s shows "SIN"
- Button t shows "COS"
- Button u shows "TAN"
- Button v shows "√x"
- Button w shows "xʸ"
- Button x shows "1/x"

## Out of scope
- Any change to button operations, shift labels, or other rows.

## Tasks
1. In `labels.json`, update the values for `key-s`, `key-t`, `key-u`, `key-v`, `key-w`, and `key-x` to "SIN", "COS", "TAN", "√x", "xʸ", and "1/x" respectively.
